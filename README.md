# BusNovaTech-EstructurasDatos
Sistema en Java para la gestión de buses y tiquetes, aplicando estructuras de datos, persistencia en JSON y atención de servicios según prioridades.

# BusNovaTech – Proyecto SC-304

## Descripción

BusNovaTech es un sistema desarrollado en Java para el curso SC-304 Estructuras de Datos.  
El objetivo del proyecto es aplicar los conceptos de Programación Orientada a Objetos y estructuras dinámicas como listas enlazadas y colas.

El sistema permite registrar usuarios, iniciar sesión y acceder a un menú principal donde se integran los diferentes módulos del proyecto.

---

## Integrantes y responsabilidades

- Integrante 1: Registro y autenticación de usuarios.
- Integrante 2: Implementación de estructuras de datos (lista enlazada y nodos).
- Integrante 3: Gestión de buses y terminales.
- Integrante 4: Integración general del sistema, menú principal y documentación.

---

## Funcionalidades actuales

- Registro de usuarios.
- Inicio de sesión.
- Carga de usuarios desde archivo JSON.
- Menú principal integrado.
- Módulo 4 con reportes:
  - Cantidad de usuarios registrados.
  - Listado de usuarios.

---

## Estructuras utilizadas

- Lista enlazada simple.
- Nodo personalizado.
- Cola para manejo de buses.
- Conversión de lista a ArrayList para generar reportes.
- Uso de JSON para guardar información de usuarios.

---

## Cómo ejecutar el proyecto

1. Abrir el proyecto en NetBeans.
2. Click derecho sobre el proyecto.
3. Seleccionar Run.
4. Elegir la clase principal:
   com.mycompany.busnovatech.Main

---

## Observaciones

El proyecto se encuentra en desarrollo y cada módulo fue trabajado por un integrante diferente.  
El módulo 4 se encarga de integrar el sistema y generar reportes básicos del estado actual.
